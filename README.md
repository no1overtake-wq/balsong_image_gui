# 발송 정리 프로그램

CSV 파일을 선택하면 상품분류명, 사입상품명, 발송수량을 기준으로 정리해서 PNG 이미지로 저장하는 프로그램입니다.

## 정리 규칙

1. `누스&리퍼브`는 무조건 맨 위에 표시
2. 나머지 상품분류는 발송수량 합계가 많은 순서
3. 상품명은 발송수량이 많은 순서
4. 발송수량이 같으면 제품명 `n` 뒤 숫자가 작은 순서
5. 마지막에 총합계 표시

## 필요한 CSV 컬럼

CSV 파일에는 아래 컬럼이 있어야 합니다.

- 상품분류명
- 사입상품명
- 발송수량

## 프로그램 사용 방법

1. 프로그램 실행
2. 큰 버튼 `발송 정리` 클릭
3. CSV 파일 선택
4. 이미지 생성 완료 후 버튼이 `다운로드`로 변경됨
5. `다운로드` 버튼 클릭
6. 저장할 폴더 선택
7. PNG 이미지 저장 완료
8. 버튼은 다시 `발송 정리`로 돌아감
